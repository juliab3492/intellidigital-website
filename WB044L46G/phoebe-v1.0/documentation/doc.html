Getting started

Choose a demo to getting started, rename that index-*.html to index.html, upload template file to web hosting.

Alway place customize style at assets/css/custom.css.
Template bootstrap.min.css has customized, do NOT replace it with cdn.

File structure

/template/
  index-*.html - template html

  /assets/
    /audio/
      bg-audio-1.mp3 - bg audio file

    /img/
      navbar-brand-media-img-dark.png   - dark text logo
      navbar-brand-media-img-light.png  - light text logo

      /about-us/
        about-us-signature.png          - #aboutUs section signature image

      /feature/
        feature-icon-*.png              - #feature section icon
        feature-media.png               - #feature section media image, for layout-3
        feature-mobile-*.png            - #feature section mobile image, for layout-1, layout-2

      /mobile-wireframe/
        mobile-wireframe-*.png          - mobile wireframe

      /page-intro/
        page-intro-bg-img.jpg                   - #pageIntro bg image
        page-intro-bg-slideshow-*.jpg           - #pageIntro slideshow image
        page-intro-bg-youtube-fallback.jpg      - youtube fallback image, mobile do not support this function, show this image on mobile
        page-intro-bg-youtube-placeholder.jpg   - youtube plaveholder image, show this image when the video not start yet

        page-intro-bg-img-pattern.png           - #pageIntro section background pattern image
        page-intro-media.png                    - #pageIntro section media image for layout 3

        service-icon-*.png              - #service section icon

      /subscribe/
        subscribe-icon.png              - subscribe icon

      subscribe.php - mailchimp subscribe form

Navbar style

Template has light or dark style and two size, we can control it by below class.

class name add to effect note
navbar-default #pageNavbar enable default style navbar n/a
navbar-inverse #pageNavbar enable inverse style navbar n/a
page-navbar-from-light-fg #pageNavbar enable transparent background and light text page-navbar before the page scroll use this class when #pageIntro is dark style
page-navbar-from-dark-fg #pageNavbar enable transparent background and dark text page-navbar before the page scroll use this class when #pageIntro is light style
navbar-lg #pageNavbar enable large page-navbar use default size by remove this class from #pageNavbar

Page Intro

Page intro size

class name add to effect group note
page-intro-sm #pageIntro enable small size #pageIntro page-intro-size n/a
page-intro-lg #pageIntro enable large size #pageIntro page-intro-size n/a
page-intro-hero #pageIntro enable hero size #pageIntro page-intro-size n/a
If no page intro option class added, page intro will use default size.

Subscribe form

Open assets/php/subscribe.php, find below code and change the setting.

// your mailchimp api key - http://admin.mailchimp.com/account/api/
$apikey = '1234512345-us10';

// your mailchimp list id - http://admin.mailchimp.com/lists/
$id     = 'abcde12345';
Please test the function at the web hosting and confirm the web hosting is support PHP.
